Лабораторная по программированию

Работа имитирует создание отеля и оперирование гостями и номерами отеля

При открытии приложения предлагается создать новый отель, задается название отеля, количество и цена номеров трех типов:

Стандартные
Люкс
Президентские

Затем открывается главное окно, где можно добавить нового пользователя вручную, либо импортировать новых пользователей из txt файла.
Также есть возможность подсчета стоимости проживания, по указанному паспорту.
При отсутствии свободных номеров или в случае ввода паспорта, который уже был добавлен, высвечивается сообщение об ошибке.
Добавлена возможность просмотра списка пользователей, проживающих в гостинице.
Для доступа к номеру паспорта пользователя необходимо ввести пароль.

Также реализовано добавление пользователей в базу данных с помощью SQLite.

Реализованы комплексные проверки полей и защита от некорректных действий пользователя. 

Программа компилируется в исполняемый EXE-файл, готовый к запуску в среде Windows.

В результате получено работоспособное, модульное и устойчивое к ошибкам ввода приложение с четким разделением логики и интерфейса.

Для запуска работы 4/5 необходимо отrрыть PPP4 > x64 > Debug > PPP4.exe

Для корректной работы программы рекомендуется использовать английский язык
